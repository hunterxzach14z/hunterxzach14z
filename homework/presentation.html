What is an Event Listener?

An event listener is a procedure or function in a computer program that waits for an event to occur; that event may be a user clicking or moving the mouse, pressing a key on the keyboard, or an internal timer or interrupt.

How to use an Event Listener.

  • Being Javascript you must use the script element
  • Then you will use this code to add an event listener.
  • element.addEventListener(event, function, useCapture);
  • You can add many event handlers to one element, and have multiple of the Same type of event
  • The addEventListener() method makes it easier to control how the event reacts to bubbling. Naturally set to false.

    • What is Bubbling?

    Bubbling the inner most element's event is handled first and then the outer.
    Capturing the outer most element's event is handled first and then the inner.
